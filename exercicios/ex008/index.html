Principais formatação

Negrito / Palavra forte

Antes utilizavasse a tag B para tornar palavra negrita (sem semântica)

No html5 utiliza-se a tag strong para tornar a palavra forte (com semântica)

Itálico / Ênfase

html4 utilizava tag i para mostrar em itálico(sem semnântica)

html5 utiliza tag em para ênfase(com semântica)

Texto Marcado

Podemos utilizar a tag de marcação com a tag MARK

Se eu quiser fazer com que todas as marcações sejam da mesma cor terei que alterar la no cabeçalho, fazendo um seletor para MARK

Texto grande e pequeno

Para texto grande não se utiliza mais a tag BIG por mais que ainda aceite

Para deixar letras pequenas, utilizamos a tag SMALL

Texto deletado

Para referir a um texto que foi excluído e não deve dar atenção a escrita utilizar a tag DEL

Texto inserido

Referente a texto que foi colocado naquele momento, utilizar a TAG INS

Texto sobrescrito

Para mostrarmos um número ao quadrado, utilizo a TAG SUP como exemplo 420

Texto subscrito

Mostrar número na parte baixa ou seja subscrito com exemplo de H2O utilizo a tag SUB
